Aggiungere cartella SMB a Jellyfin

  1. Montare in locale la cartella SMB:
  • sudo mkdir /mnt/nas
  • sudo apt-get install cifs-utils
  • sudo mount -t cifs -o user=plraska //192.168.1.254/iliadbox/Video /mnt/nas
  1. Creare file .smb nella home con le credenziali di accesso:
  • username = plraska
  • password = Cuor_di_mele

  1. Aggiungere queste righe al file /etc/fstab //192.168.1.254/iliadbox/Video /mnt/nas cifs uid=0,credentials=/home/plraska/.smb,iocharset=utf8,vers=3.0,noperm 0 0
